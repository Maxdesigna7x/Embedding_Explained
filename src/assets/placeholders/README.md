# Placeholders

La presentacion usa formas, texto y emojis, por lo que no requiere imagenes externas para funcionar.

Si despues se quieren reemplazar objetos por assets reales, colocar aqui:

- `object_text.png`: representacion visual de texto.
- `object_image.png`: representacion visual de imagen.
- `object_audio.png`: representacion visual de audio.
- `object_document.png`: representacion visual de documento.
- `model_box.png`: ilustracion opcional de modelo encoder.

Actualizar los componentes `ObjectCard` o las slides correspondientes para usar estas imagenes.
